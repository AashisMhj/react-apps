# E commerce Dashboard
A E-commerce page to show case the use of react-query.
Data fetched from https://fakestoreapi.com.

## Technologies
* React
* Material Ui
* react query

## Prerequisite
* node
* a package manager
* stripe account

## Installation guide
```bash
# create a .env file and copy the content of .env.example

## install packages
yarn

## run dev server
yarn dev
```
